# Software Architecture

Use this skill for software system design and review.

## Modes

- `/sa design`
- `/sa review`
- `/sa ddd`
- `/sa clean-architecture`
- `/sa clean-coding`
- `/sa clean-ai`
- `/sa adr`
- `/sa integration`
- `/sa diagram`

## Diagrams

Use `/sa diagram` for C4, sequence, component, deployment, Clean Architecture,
API, event, data ownership, and Clean AI software diagrams in Mermaid,
PlantUML, or C4-style notation.
