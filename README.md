# software-architecture-skill

Software Architecture skill for system design, service boundaries, Domain-Driven
Design, Clean Architecture, Clean Coding review, ADRs, integration patterns,
quality attributes, and Clean AI design.

## Install For Codex

```powershell
npx -y skills add MartinEgli/software-architecture-skill --skill * -a codex --yes
```

## Checks

```powershell
npm run validate
npm run test
npm run package
```
